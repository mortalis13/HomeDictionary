Home Dictionary is a dictionary shell application that allows to connect multiple external dictionary files for any language.

It supports Lingvo DSL dictionary format which is a plain text with special syntax markup that maps words/phrases to their translation articles.

The DSL text files can be compressed to DSL.DZ format to save disk space, using the dictzip compression tool. The application recognizes .DZ file format and reads dictionary contents from it.

Features
  • word search in multiple dictionary files on a selected translation direction
  • search of word translation to/from multiple languages/directions (e.g. De->(*), (*)->En)
  • change order of dictionaries in the found article
  • search history filtered by translation direction and ordered by searched word, direction, number of searches or search date
  • selectable main directory for dictionary files scanning
  • support for abbreviation dictionaries
Usage
1. Scan Dictionaries
  • put DSL or DSL.DZ files in a folder on your filesystem
  • open Scan Dictionaries panel and press Scan
  • during the scanning the scanned number and total number of dictionaries will be shown
  • successfully scanned dictionaries will have green progress bar
  • if some dictionary files are incorrect the progress will be red and the number of Errors at the top will increment, you can select Show Errors in the menu to view list of dictionaries with error messages
  • you can skip some dictionaries during the scanning by selecting them or use Activate/Deactivate All menu to change scan status for all dictionaries
  • you can cancel scanning process, the scanned dictionaries until that moment will not be lost
  • during the scanning you can search words in dictionaries that you already have scanned previously, just swipe to the left to go to the main panel
  • the Re-Scan menu updates list of language pairs based on current contents of the main folder with dictionaries (used when dictionaries are deleted from the application or manually from the filesystem and the application should reflect the correct language list)
2. Word Search
  • select a translation pair from the list on the right
  • use the arrows button to switch languages, if it's enabled
  • pairs of type "lang->(*)" or "(*)->lang" allow to search translations from/to a selected language to/from all available languages
  • enter a word in the text field and select a result in the list below
  • max number of results can be selected in the Settings, higher values will require more time for the search process
  • use the search button/Enter on your soft keyboard to select the first result in the list
3. Article Navigation
  • article content is mainly styled based on the formatting tags of the DSL files
  • article zoom is smooth on older devices (versions 4.1.2+) and incremental on other (changes zoom level when zoom gesture is released), the zoom increment can be changed in the Settings, as well as default font size
  • to zoom with one finger doubleclick on the article content and move finger up or down
  • for large articles the scrollbar thumb can be used to navigate fast through the article content
  • the order of dictionaries in the article can be changed in the Edit Dictionaries menu
  • the Search Text menu is used to find a text inside the article content
  • some articles contain links (in green), clicking on them will load the corresponding articles
  • pressing the Back button on the device you go to the search list of the current word
  • clicking on the word text field will search for that word again and show the search result list
4. Edit Dictionaries
  • to change the order of dictionaries in the article select them and use up/down arrows to move the items
  • use Delete button to remove dictionary from the search, it will not remove the corresponding physical file so you can scan it again
  • long press a dictionary to see its details
5. Search History
  • the history list shows the searched word, number of times the article was opened and the translation direction
  • clicking on a word will load an article for it
  • the list can be ordered by word text, search direction, number of hits and date/time
  • the direction selector includes all language pairs as in the main view and also has "--All--" item for all directions history
6. Abbreviation Dictionaries
  • these dictionaries use the same DSL format but should have "_abrv" at the end of the filename
  • they are rendered as tables of abbreviation-meaning pairs
  • scanned from the Abbreviations menu with the Rescan button
  • the Back toolbar button is used to return to the abbreviation dictionary list from the table view
7. Settings
  • the "Main Directory" option allows to select a folder with dictionaries to scan
  • it's possible to switch between folders that previously were selected, it will load scanned dictionaries, order of dictionaries and search history, so it's not necessary to scan the dictionaries again if they were previously scanned
  • the "Max Results" limits maximum number of words searched and shown in the result list
  • the "Include target=source" is used to search words in the Explanatory types of dictionaries (where source language is the same as the target one), e.g. if you search a word in [en-de] direction the application will also search the [en-en] dictionaries if available
  • the "Detect source language" option changes the translation direction based on the first letter of the search word (if it's a latin letter, cyrillic, etc.)
  • the "Article Font Size" is for the default font size, it will be reset to this value when the article was zoomed
  • the "Article Font Increment" determines the value the font increases/decreases when article content is zoomed in/out
Notes
  • use of external sdcard is restricted to devices and Android versions that allow write access for that type of storage. For example, on devices with Android KitKat (4.4) external storage is read-only by default, to make it writable the device should be rooted, then you can use the SDFix: KitKat Writable MicroSD application to fix the default permissions
