How To Compress DSL to DSL.DZ

Raw DSL files may occupy a lot of disk space depending on the number of translated articles and file encoding.

It's useful to compress this type of files and to be able to access its content without unpacking the compressed archive.

dictzip tool is used for such a compression, it creates a .dz file and allows random access to the files inside it.


For Windows it can be downloaded as a binary executable from this repository: https://github.com/Tvangeste/dictzip-win32/releases.

Other implementations: https://github.com/dictzip.


To compress a DSL file use this command:


  dictzip -k some_dict.dsl

It will create a file named "some_dict.dsl.dz". It's an archive file which can be uncompressed with 7zip program.
