How To Convert Other Dictionary Formats to DSL
Convert from StarDict to DSL
1. Download the stardict_to_dsl tool and unpack the .zip archive.
2. Launch the command line window (press Win+R -> type "cmd" -> Enter) and execute the following command:

  stardict_to_dsl "c:\latin-english.ifo"
3. This will create a .dsl file in the same folder as the source file.

Convert from BGL to DSL
1. Download the bgl_to_dsl tool and unpack the .zip archive.
2. Launch the command line window (press Win+R -> type "cmd" -> Enter) and execute the following command:

  bgl_to_dsl "c:\Babylon_Dict.bgl"
3. This will create a .dsl file in the same folder as the source file.
